Типографіка
Три шрифтові сімейства з чіткими ролями. Display для заголовків, Serif для акцентів і цитат, Sans для інтерфейсу.
Playfair Display
--display · Заголовки H1–H2, ціни, великі числа
Архітектура мислення, рішень і масштабу
Italic для акцентних слів
Cormorant Garamond
--serif · Цитати, підзаголовки, великий акцентний текст
Іноді саме одна розмова стає точкою перелому у житті та бізнесі людини.
Raleway
--sans · UI текст, кнопки, лейбли, параграфи
Допомагаю людині побачити головне, прибрати зайве і повернутися до своєї сили, ясності та масштабу. Бо часто наступний рівень життя починається не з нової стратегії.

Символ бренду
SVG-символ використовується як Three.js particle shape, preloader animation, фонові декоративні елементи.
/* Використання символу */
Preloader: stroke-dasharray animation (малювання)
Three.js: SVG → offscreen canvas → pixel sampling → point cloud
Фон: opacity .03–.04, slow rotation 60–80s
Final CTA: breathe animation (scale + opacity pulse)
